# Guía de estilos y recursos visuales – TuberIA

Este directorio recoge la definición visual base para el diseño de TuberIA. Incluye referencias de color, tipografía, componentes iniciales y bocetos pensados para experimentar el aspecto y validar la integración de cada elemento en la interfaz principal.

---

## Descripción general

Los recursos aquí listados permiten probar y justificar el aspecto funcional y estético del producto, antes de avanzar a prototipos avanzados o desarrollo. Todos los materiales están orientados a cubrir el alcance y requisitos del MVP y se adaptan rápido a nuevas necesidades.

---

## Recursos incluidos

1. **Paleta de colores**
    - Esquema visual con los tonos principales y secundarios utilizados en la app.

2. **Tipografías y jerarquías**
    - Ejemplo de fuentes y tamaños para cada nivel de texto (títulos, subtítulos, cuerpo) y orientaciones de uso básico.

3. **Componentes UI**
    - Botones, campos de entrada, tarjetas y primeros prototipos para testear visualmente la integración en pantallas clave.

4. **Bocetos/wireframes rápidos**
    - Vistas esquemáticas que muestran cómo se combinan los elementos principales en las diferentes secciones de la app.
