# Wireframes MVP – TuberIA
Este directorio contiene los wireframes digitales realizados para definir el diseño funcional y validar los flujos principales del MVP (Producto Mínimo Viable) de la aplicación TuberIA.

## Descripción general
Los wireframes representan de manera clara las pantallas principales, siguiendo la definición y alcance del **MVP** acordados por el equipo. Cada diseño replica el aspecto y la navegación esperada en la versión inicial del proyecto.

## Pantallas incluidas
1. **Login / Registro**

    - Pantalla central para el acceso y alta de usuario.

2. **Búsqueda y seguimiento de canales**

    - Barra de búsqueda destacada.

    - Resultados de búsqueda con botón para seguir canales.

    - Sección visual separada para los canales que el usuario ya sigue.

3. **Resúmenes (Panel principal)**

    - Muestra los resúmenes generados por IA, agrupados por canal seguido.

    - Tarjetas individuales para cada vídeo con resumen y acceso a detalle.

4. **Detalle de vídeo resumido**

    - Visualización ampliada: miniatura y título del vídeo, resumen generado por IA y transcripción completa.

    - Botón para regresar al panel principal.


## Entrega
Archivos en formato PNG, nombrados por pantalla.

Este README documenta la organización y el propósito de cada imagen entregada.
